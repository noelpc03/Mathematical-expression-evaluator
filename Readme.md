# Evaluador de Expresiones Matemáticas

Este proyecto es un evaluador de expresiones matemáticas desarrollado en Haskell, con una interfaz gráfica que permite a los usuarios ingresar y evaluar expresiones en tiempo real. 

## Características Principales

- **Aplicación web**: Desarrollada con Threepenny-GUI para la interfaz de usuario.
- **Evaluación de expresiones matemáticas**: Soporta operaciones básicas, así como también integración y derivación.
- **Soporte para números decimales, binarios y hexadecimales**: Conversión entre bases y evaluación de expresiones en estos formatos. 

## Instalación

Para ejecutar el proyecto, asegúrese de tener instalado GHC y Cabal. Luego, siga estos pasos:

1. Clonar el repositorio:
   ```sh
   git clone https://github.com/noelpc03/Mathematical-expression-evaluator
   cd Mathematical-expression-evaluator
   ```

2. Ejecutar el script de configuración:
   ```sh
   ./setUp.sh
   ```

3. Ejecutar el proyecto:
   ```sh
   ./run.sh
   ```
